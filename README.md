# dotclaude

Analyze and visualize your Claude Code usage patterns.

## Install

```bash
pipx install dotclaude
# or
uv tool install dotclaude
```

## Usage

```bash
dotclaude                    # terminal dashboard
dotclaude --json             # JSON output
dotclaude --html report.html # HTML report
dotclaude --insights         # usage health check
dotclaude --evolve           # config recommendations
```
